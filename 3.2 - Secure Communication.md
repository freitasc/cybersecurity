## What is it?

- Ways to securely connect to a secure network
- VPNs
## Additional Information

- VPN (Virtual Private Network)
	- Encrypted data traversing a public network
	- Concentrator
		- Encryption/decryption access device
		- Often integrated into a firewall
	- Many deployment options
		- Specialized cryptographic hardware (usually for corporations)
		- Software-based options available (small business or common users)
	- Used with client software
		- Sometimes built into the OS
