## What is it?

- Security risks are out there
	- Many different categories and types to consider
- Assets are also varied
	- Data, physical property, computer systems
- Prevent security events, minimize the impact and limit the damage

## Additional Information

- #### CONTROL CATEGORIES
	- Technical Controls
		- Controls implemented using systems
		- Operating system controls
		- Firewalls, anti-virus, EDR / XDR
	- Managerial Controls
		- Administrative controls
		- Security policies
		- Security documentation
		- Standard operation procedures
	- Operational controls
		- Controls implemented by people
		- Security guards, awareness programs
	- Physical Controls
		- Limit physical Access
		- Fences, locks, badge readers

- #### CONTROL TYPES
	- Preventive
		- Block access to a resource
		- Firewall rules
		- Door locks
		- Identification check on building entry
	- Deterrent
		- Discourage an intrusion
		- robot.txt, app splash screens, user legal policies
		- Lighting
		- Front reception desk
		- Warning signs
	- Detective
		- Identify and log an intrusion attempt
		- System Logging
		- Patrolling the property
		- Motion detectors
	- Corrective
		- Apply a control after an event has been detected, reversing the damage
		- Restoring backups
		- Creating policies
		- Contact law enforcement
		- Fire extinguisher
	- Compensating
		-  Control using other means, may be a temporary mesure
		- Firewall blocks specific app instead of patching the app
		- Implement a separation of duties
		- Require simultaneous guard duties
		- Generators for power outage cases
	- Directive
		- Direct a subject towards security compliance
		- ==The weakest security control==
		- "Do this, please"
		- Store all sensitive files in a protected folder
		- Create compliance policies and procedures
		- Train users on proper security policy
		- Post a sign for "Authorized personnel only"
