## What is it?

- Where you are compared with where you want to be
- This may require extensive research
- This can take weeks or months

## Additional Information

- Work towards a known baseline (or framework)
	- This may be a framework, internal policy or set of goals
	- More serious organizations should use formal standards
- Determine the end goal
	- NIST 800-171 Revision 2
	- ISO/IEC 27001
	- etc
- Get a baseline of employees
	- Formal experience, current training
	- Knowledge of security policies and procedures
- Examine the current process
	- Enumerate IT systems
	- Evaluate existing policies and protocols
	- Compare systems
	- Identify weaknesses
	- Create a detailed analysis
		- Examine the broad security categories
		- Break those into smaller segments
- Analysis and report
	- Detail baseline objectives
	- Create a clear view of the current state
	- Make a path to get from current security to the goal
		- This should include time, money and changes
	- Time to create the gap analysis report
		- A formal description of the current state
		- Recommendations for meeting the baseline
