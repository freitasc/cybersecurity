## What is it?

- Cloud is a very complex ecosystem and for that reason attackers have great opportunities
## Additional Information

- Attacking the services
	- DDoS
		- Most of the times the cloud has a static public IP issued by the vendor of the platform
	- Authentication Bypass
	- Directory traversal
	- Remote code execution (RCE)
- Attacking the application
	- Cross-site scripting
	- Out of bounds writing
		- Writing in unauthorized memory areas
		- Can crash or corrupt a server, exploiting it for gain is a hard task
	- SQL injection
