## What is it?

- SQL is the most common relational database management system language in the world
- SQL injection is the act of injecting your own SQL requests into an existing application

## Additional Information

- Can and often is executed from a web browser
	- injected in a form, field, search bar or url
	- this could be very bad
	- could be use to view all the database content, delete database information, add and remove users, DoS, etc
