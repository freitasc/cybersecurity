## Additional Information

- Patching
	- Incredibly important
	- Monthly updates
	- Third-party updates
	- Auto-update
	- Emergency updates
- Encryption
	- Prevent access to app data files
	- File level encryption
	- Full disk encryption
	- App data encryption
- Monitoring
	- Aggregate info from devices (SIEM)
	- Sensors (IPS/IDS, EDR/XDR)
	- Collectors
- Least privilege
	- Rights and permissions should be set to minimum
	- All user accounts must be limited
	- Don't allow users to run with admin privileges
- Configuration enforcement
	- Perform a posture assessment
	- Extensive check
		- OS patch
		- EDR/XDR version
		- Status of firewall
	- System out of compliance are quarantined and often segmented
- Decommissioning
	- Should be a formal policy
	- Mostly associated with storage
	- Many options of physical devices
		- Recycle the device for use in another system
		- DESTROY IT.
