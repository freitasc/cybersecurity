## Additional Information

- Physical Isolation
	- For example: air gap between Switch A and Switch B
		- No opportunity for mixing Data A with Data B
	- Must be connected to provide communication
	- Web servers in one rack, Database serves on another
- Logical Segmentation with VLANs
	- Virtual Local Area Networks (VLANs)
	- Separated logically instead of physically
	- Cannot communicate between VLANs without a Layer 3 device / router
- SDN (Software Defined Network)
	- Networking devices have different functional planes of operation
		- Data, control and management
	- Split the functions by separate logical units
		- Perfectly built for the cloud
		- Extend the functionality and management of a single device
	- Infrastructure Layer / Data Plane
		- Process the network frames and packets
		- Forwarding, trunking, encryption, NAT
	- Control Layer / Control Plane
		- Manages the actions of the data plane
		- Routing tables, session tables, NAT tables
		- Dynamic routing protocol updates
	- Application Layer / Management Plane
		- Configure and manage the device
		- SSH, Browser, API
