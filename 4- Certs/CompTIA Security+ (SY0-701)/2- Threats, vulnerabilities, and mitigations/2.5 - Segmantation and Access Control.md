## What is it?

- Physical, logical or virtual segmentation
- Performance
- Security
- Compliance
	- PCI DSS obliges segmentation
## Additional Information

- Access control lists (ACLs)
	- Allow or disallow traffic
		- Groupings of categories
		- Source IP, Destination IP, port, time of day, app etc
	- Restrict access to network devices
		- Limit by IP addrewss or other identifier
		- Prevent regular user / non-admin access to reach certain things
- Application allow list / deny list
	- Any application can be dangerous
		- Vulnerabilities, trojan horses, malware
	- Security policy can control app execution
		- Allow list, deny/block list
	- Allow list = nothing run unless it's approved
	- Deny list = everything runs except what is unapproved
