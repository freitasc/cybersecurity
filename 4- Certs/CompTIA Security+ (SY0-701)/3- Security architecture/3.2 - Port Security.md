## What is it?

- Refers to the security in the individual interfaces there are on switches
- Also refers to the abstracted interfaces on wireless access points
## Additional Information

- EAP (Extensible Authentication Protocal)
	- An authentication framework
	- Many different ways to authenticate based on RFC standards
		- Manufacturers can build their own EAP methods
	- EAP integrates with 802.1X
		- Prevents access to the network until the authentication succeds
- IEEE 802.1X
	- Port-based Network Access Control (NAC)
	- You don't get access to the network until you authenticate
	- EAP integrates with 802.1X
		- 802.1X prevents access to the network until the authentication succeeds
- Used in conjunction with an authentication database
	- RADIUS, LDAP, TACACS+, Kerberos, etc.
- IEEE 802.1X + EAP
	- Supplicant - the client
	- Authenticator - the device that provides access
	- Authentication server - Validates the client credentials
