## What is it?

- Match supply to the demand
- This isn't always obvious
## Additional Information

- People
	- Some services require humans
		- Call center
		- Tech services
	- Too few and things don't have the same quality
	- Too many and it becomes a burden
- Technology
	- Pick a technology that can scale
	- Web services
		- Load balancers and multiple web services
		- Microservices are the new hot thing
	- Database services
		- Cluster multiple SQL services
	- Cloud services
		- Services on demand
		- Seemingly unlimited (but the cost also scales drastically)
- Infrastructure
	- The underlying framework
		- App servers, network services
		- CPU, network, storage
	- Physical devices
		- Purchase, configure, and install
	- Cloud-based devices
		- Easier to deploy
		- Useful for unexpected capacity changes
