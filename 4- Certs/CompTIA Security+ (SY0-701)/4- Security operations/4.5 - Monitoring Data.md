## Additional Information

- FIM (File Integrity Monitoring)
	- Some files change all the time, and some should never change
	- Monitor important operating system and application files
	- Windows -> SFC (System File Checker)
	- Linux -> Tripwire
	- Many host-based IPS options
- DLP (Data Loss Prevention)
	- Stop the data before the attackers get it
	- So many sources, so many destinations
	- On your computer = Data in use
	- On your network = Data in motion
	- On your server = Data at rest
	- Cloud-based DLP
		- Located between users and the internet
		- Block custom defined data string
		- Manage access to URLs
		- Block virus and malware as well
	- DLP and email
		- Check every email inbound and outbound
		- Inbound
			- Block keywords, identify impostors, quarantine emails
		- Outbound
			- Fake wire transfers, W-2 transmissions, employee information
- USB blocking
