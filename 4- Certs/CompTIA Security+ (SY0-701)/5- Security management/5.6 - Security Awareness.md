## What is it?

- Training people so they don't make mistakes as often

## Additional Information

- Phishing campaigns
	- Many companies will perform internal phishing campaigns
	- Can be automated
	- They are done to check if employees can recognize and report a phishing attempt
- Anomalous behavior recognition
	- Employees can be seen doing risky behavior
		- Modifying hosts file
		- Replacing a core OS file
	- Unexpected behavior
		- Logon from another country
		- Increase in data transfers
	- Unintentional behavior
		- Typing the wrong URL
		- Misplacing USB drives
		- Misconfiguring security settings
- Reporting and monitoring
	- Track and analyze security awareness metrics
	- Initial vs Recurring
		- A first nonoccurence is opportunity for user training
		- High frequency is a bad sign
- Development
	- Create a Security Awareness Team
	- Establish a minimum awareness level
	- Integrate compliance mandates
	- Define metrics
- Execution
	- Create the training materials
	- Document success measurements
	- Identify the stakeholders
	- Deploy the training materials
	- Track user training efforts
